User Guide

Application: EasyMp3Play
Effective Date: 2025-09-04
Last Updated: 2025-09-12

1. Folder Selection & Supported Formats

On first launch, tap Select Folder and choose a music directory.
Supported files: MP3, M4A, AAC, WAV, FLAC, OGG, OPUS.

2. Playback Controls

  • Play/Pause – start or stop playback.
  • Previous / Next – skip tracks.
  • -60s, -10s, +10s, +60s – seek backward or forward.

3. Shuffle, Repeat & Speed

  • Shuffle – toggle random playback.
  • Repeat – cycle Off → All → One.
  • Speed – cycle 1.00× → 1.25× → 1.50× → 1.75× → 2.00×.

4. Volume

Use Volume + to raise and Volume - to lower the system volume.

5. Equalizer & Presets

Tap Equalizer to open a 5-band EQ.
Adjust sliders or apply presets: Bass, Flat, Vocal.

6. Sleep Timer

Tap Sleep to set a timer (15–120 min) that stops playback automatically.

7. Menu Options

Tap the Menu button (≡) to change the Color Theme, view the License, or read the Privacy Policy.
