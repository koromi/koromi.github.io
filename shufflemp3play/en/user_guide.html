User Guide

Application: ShuffleMp3Play
Effective Date: 2025-09-04
Last Updated: 2026-01-10

1. Folder Selection & Supported Formats

On first launch, tap Select Folder and choose a music directory.
Supported files: MP3, M4A, AAC, WAV, FLAC, OGG, OPUS.

The app searches the selected folder and every subfolder, so album and playlist folders can stay nested without leaving tracks out of the library.

Important: If you add, rename, or delete audio files inside the selected folder, use Menu > Select Folder to pick the directory again (or restart the app). This refreshes the library so the new changes appear in the track list.

Quick subfolder search

Tap the Search button to toggle subfolder search. When Search mode is active, the button label reads Search; in normal mode the label reads Off.

2. Playback Controls

  • Play / Pause – start or stop playback.
  • Previous / Next – skip tracks.
  • -60s, -10s, +10s, +60s – seek backward or forward.

3. Shuffle, Repeat & Speed

  • Shuffle – toggle random playback.
  • Repeat – cycle Off → All → One.
  • Speed – 1.00× and 2.00× are free. 1.25×–1.75× require the free trial or Premium.

4. Equalizer & Presets

Tap Equalizer to open a 5-band EQ.
Everyone can apply presets: Bass, Flat, Vocal; moving the sliders manually requires the free trial or Premium.

5. Sleep Timer

Tap Sleep to set a timer that stops playback automatically. The 15-minute option is free; choosing 30–120 minutes requires the free trial or the Premium upgrade.

6. One-Time Wake Up

Use One-time Wake Up to pick a time for ShuffleMp3Play to resume or start playback later. The schedule is stored locally on the device and can be canceled from the same dialog. This feature is available during the free trial and with the Premium upgrade.

  • The feature uses exact alarms; if the OS blocks exact alarms, the wake time will not run. Allow the app to schedule exact alarms if prompted.
  • After rebooting or changing the device time/zone, the app attempts to restore pending wake times while it is allowed to receive system broadcasts. If the wake time is already in the past, it is cleared.
  • If playback is already running when the wake time arrives, the app leaves the current playback as-is.

7. Menu Options

Tap the Menu button (≡) to access:

  • Color Theme – adjust the player's accent colors.
  • User Guide – view the user guide inside the app.
  • License – view license information.
  • Privacy Policy – view the privacy policy.
  • Premium upgrade – open the purchase screen for Premium details and upgrade options.
  • Exit – stop playback and close the app.

Playback progress is always shared between folder browsing and subfolder search; resume keys are kept in sync automatically.

8. Bluetooth Playback Notes

When using Bluetooth audio devices, slight volume wobble or momentary dropouts may occur about one second after playback starts. These effects can depend on the Bluetooth device model or communication profile limitations. If they become disruptive, try reconnecting the device or switching to a different output.

9. Background Playback Tips

If playback stops, pauses, or briefly mutes while the screen is off or after long listening sessions, the device's power management may be limiting background activity. To keep ShuffleMp3Play running smoothly:

  • Review the system's battery optimization or power saver settings and allow ShuffleMp3Play to continue running without restrictions.
  • If your device offers manual control over automatic app launches or background protection lists, ensure ShuffleMp3Play is permitted to stay active.
  • Restarting the device can clear resource pressure that accumulates over long uptimes and restore stable playback.
